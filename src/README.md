# Source Directory

This directory contains the main application source code.

## Modules

- **FigmaClient.js** - Figma API integration wrapper (Requirements 1.1, 1.2, 1.3)
- **GitHubClient.js** - GitHub API integration wrapper (Requirements 4.1, 4.2, 4.3, 4.4)
- **RateLimiter.js** - Rate limiting utility for API requests (Requirements 3.1, 3.2, 3.3)

## Usage

Each module exports functions that can be imported and used in the extraction service and API server.
